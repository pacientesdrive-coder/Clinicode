# ClinCoord Mental — proyecto listo

Este paquete ya trae tu app React preparada como proyecto Vite.

## Para verla en tu computador

1. Instala Node.js desde https://nodejs.org/
2. Abre una terminal dentro de esta carpeta.
3. Ejecuta:

```bash
npm install
npm run dev
```

4. Abre la dirección que aparece, normalmente http://localhost:5173/

## Para crear una versión publicable

```bash
npm run build
```

Eso crea una carpeta llamada `dist`.

## Nota importante

Esta versión usa datos de demostración dentro del frontend. Todavía no tiene base de datos real, login real ni seguridad clínica de producción.
